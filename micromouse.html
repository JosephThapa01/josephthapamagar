üìã About This Project

Built a fully autonomous Micromouse robot that can find its way through a 16√ó16 maze, reach the center, and navigate quickly using smart algorithms and precise motor control. The project demonstrates the integration of path-finding algorithms, real-time control systems, and custom hardware design to achieve efficient autonomous maze navigation.

Key Contributions

Algorithm Development

  • Implemented a modified flood-fill algorithm optimized for memory efficiency, enabling the robot to find the shortest path through complex maze configurations.
  • Developed dynamic path updating system that adapts to newly discovered walls during exploration.

Control Systems

  • Created a PID/PD motor control system enabling smooth wall-following behavior and precise turning maneuvers.
  • Conducted comprehensive motor analysis in MATLAB to characterize system dynamics and optimize controller parameters.
  • Implemented real-time encoder signal processing to maintain accurate position tracking throughout navigation.

Simulation and Testing

  • Established simulation environment in ROS + Gazebo with custom robot models to validate navigation algorithms before hardware deployment.
  • Tested across multiple maze configurations to verify shortest-path accuracy and robustness.
  • Collected and analyzed sensor data to validate accuracy and identify areas for improvement.

Hardware Architecture

Custom Electronics

  • Custom PCB Design: Designed and fabricated custom printed circuit boards integrating all electronic components for compact and reliable operation.
  • STM32 Microcontroller: High-performance microcontroller executing real-time control algorithms and sensor fusion.
  • L293D H-Bridge Driver: Dual H-bridge motor driver providing smooth bidirectional motor control with current protection.

Sensing System

  • VL53L0X TOF Sensors: Time-of-flight laser ranging sensors providing reliable wall detection with millimeter accuracy.
  • Magnetic Encoders: High-resolution rotary encoders integrated with motors for precise distance and velocity feedback.
  • Sensor fusion algorithms combine TOF and encoder data for robust localization within the maze.

Mechanical Design

  • Compact chassis design optimized for navigating standard Micromouse maze dimensions (180mm cells).
  • Low center of gravity configuration ensuring stability during high-speed runs and sharp turns.
  • Integrated motor mounts and sensor brackets for consistent alignment and measurement accuracy.

Software & Algorithms

Maze Mapping and Pathfinding

  • Real-time maze mapping system builds internal representation as the robot explores.
  • Modified flood-fill algorithm computes optimal paths with minimal computational overhead.
  • Dynamic path recalculation adapts to newly discovered obstacles during exploration phase.

Motion Control

  • Cell-to-Cell Navigation: PID controller ensures precise movement between maze cells with minimal overshoot.
  • Wall Following: PD control maintains optimal distance from maze walls during exploration.
  • Turn Management: Coordinated motor control executes 90¬∞ and 180¬∞ turns with repeatable accuracy.
  • Speed Profiling: Velocity control optimizes speed while maintaining positional accuracy.

Real-Time Processing

  • High-frequency control loop (1kHz+) processes encoder signals and updates motor commands.
  • Interrupt-driven sensor readings ensure timely wall detection and collision avoidance.
  • State machine architecture coordinates exploration, pathfinding, and speed run phases.

Competition Performance

  • First Runner-Up at IIT Bombay Techfest 2022: Successfully competed against teams from across India in the Micromouse challenge.
  • Demonstrated consistent maze solving across multiple runs with different maze configurations.
  • Achieved reliable shortest-path calculation and execution within competition time constraints.

Future Enhancements

Advanced Motion Planning

  • Diagonal movement capability for faster maze traversal using 45¬∞ angles.
  • Smooth curve navigation to replace discrete 90¬∞ turns, reducing time loss during direction changes.

Hardware Improvements

  • Suction fan integration to increase downforce and improve traction at higher speeds.
  • FPGA-based control implementation for ultra-fast control loops and parallel processing.
  • Upgraded motors and encoders for higher speed and resolution.
